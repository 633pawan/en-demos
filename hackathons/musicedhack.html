The Echo Nest API is the Ultimate Music API

The Echo Nest API provides broad and deep data on millions of artists and songs, providing the perfect technology platform to help you transform music education in NYC schools. To get started, register for an Echo Nest developer account and get yourself an API key. It will take less than a minute.

Get your API Key

Getting Started

Once you've created an account and have received your API key, check out some of our demos, take a look at the tutorials, get inspiration from the labs, download one of our client libraries and start building cool stuff.

Tutorials

The best way to understand an API is by diving in and seeing working examples in action. Each of our tutorials shows the exact calls to make to achieve a particular result and includes clickable examples allowing you to execute the calls in your browser and see the response. Visit the Tutorials.

The Labs

When we build cool stuff, we put it on our Labs Page Check out the Infinite Jukebox, Boil the Frog and The Bonhamizer to get a taste of what you can do with The Echo Nest.

Demos

We have lots of demonstration apps that show you how to do just about anything possible with The Echo Nest API, from retrieving basic artist data, to advanced personalized, dynamic playlisting. Check out the demos at the The Echo Nest Demo Page
